# Flutter Hello World

## Setup

1. **Install Flutter**
   It seems Flutter is not installed on your system. You can install it using Homebrew:
   ```bash
   brew install --cask flutter
   ```
   *Note: Please ensure you have sufficient disk space (at least 10GB recommended) as the current available space (3.3GB) is insufficient.*

2. **Initialize Project**
   Since the project was created manually, you need to generate the platform-specific files:
   ```bash
   cd flutter_hello_world
   flutter create .
   ```

3. **Run the App**
   ```bash
   flutter run
   ```
